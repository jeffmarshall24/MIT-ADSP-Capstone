Used Cars Price Prediction Report

1. Problem Definition

The Context

Used car price prediction is critical because it empowers buyers, sellers, and financial institutions to make informed decisions. Accurate pricing leads to fair transactions, reduced information asymmetry, and a more efficient used car market. For example, dealerships can optimize inventory, while banks can better assess loan risks.

The Objective

The main goal is to build a robust regression model that accurately predicts the used car price (in INR 100,000). This model will help stakeholders determine fair market prices, streamline negotiations, and improve market transparency.

The Key Questions

  • What factors influence used car prices? (e.g., Year, Kilometers driven, Fuel_Type, Transmission, etc.)
  • How do numerical and categorical features relate to price? (Are variables like Year and Mileage significant predictors?)
  • Is data transformation (e.g., log transformation) necessary? (Does the distribution of the price variable require normalization?)
  • Can feature engineering improve predictions? (For instance, can we extract the brand name from the Name column to reduce high cardinality?)

The Problem Formulation

Using data science, we aim to solve a regression problem where the target variable is the used car price. Our approach involves data cleaning, exploratory data analysis (EDA), feature engineering, model building, hyperparameter tuning, and performance evaluation.

  • Observations:

Refined Insights

  • Key Predictors:

  • Model Behavior:
  • Log transformation of the target helped normalize the distribution, leading to improved predictions.
  • Data Considerations:
  • Removing high cardinality features (e.g., Name) prevents overfitting and leakage.
